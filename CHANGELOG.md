# Changelog

All notable changes to this project will be documented in this file.  

---
## Full Release - 1.1.0 - 29 March 2026

### Changed

- Enscriber's core has been updated to handle ordering and messages better
- Sending indicator has been fixed
- Incompatibility warnings have been added
- Emotes are no longer guarded by throttler's prompt guard
- Some colors and styling has been updated
---

## Hotfix - 1a2b - 26 March 2026

### Changed
- Added "Show Lockdown Notifications" Toggle to show or suppress lockdown notifications.
- Added Lockdown Indicator for better on-screen clarity of encounter lockdowns.
- Added detection for leaving instance incase user leaves instead of killing boss to end encounter.
- Added Native UI Settings Window
- Changed how Enscriber handles messages in channel 1 (say)

### Fixed
- Resolved an issue causing messages to be lost when under abnormal latency.

### Maintenance
- Deprecated Compatibilities
- Updated Enscriber Core
- Updated TOC Version
